## Skills Demonstrated Across Portfolio

**SQL (MySQL)**
- Window functions (`ROW_NUMBER`, `RANK`, `LAG`, `LEAD`, `NTILE`)
- Common Table Expressions (CTEs) — multi-step, recursive, chained
- Joins across multiple sources (INNER, LEFT, CROSS, self-joins)
- Subqueries and correlated subqueries
- Aggregations, `GROUP BY`, `HAVING`, `ROLLUP`
- Stored procedures and views
- Data cleaning, normalization, and validation logic
- Composite scoring and weighted index models

**Data Visualization (Tableau)**
- Choropleth / filled maps at ZIP and county level
- Executive KPI dashboards
- Trend lines, bar/waterfall charts, scatter plots
- Risk tier segmentation visuals
- Dual-axis charts and calculated fields

**Analytics & Storytelling**
- Epidemiological disparity analysis
- Value-based care population segmentation
- Claims reimbursement and denial pattern analysis
- Operational KPI variance reporting
- Written executive summaries and consulting-style memos
- Research-grade findings with recommendations
